脚本使用

1. 下载 chrome/chromium 浏览器的 tempermonkey 插件
2. 安装脚本[脚本地址](https://greasyfork.org/zh-CN/scripts/530831-better-miyoushe)

源码使用

```shell
pnpm install

pnpm dev
```

构建项目

```shell
pnpm build
```

blob-html 需要单独构建使用

本项目仅供学习参考使用，最终解释权归米哈游所有
